## Wasp Cheatsheet

This app is configured with the TypeScript Wasp Spec API in `main.wasp.ts`. The Wasp version is listed there.

- Use `https://wasp.sh/llms.txt` as the Wasp API reference, check it before writing code.
- Import Spec constructors from `@wasp.sh/spec` and export `app({ ... })` as the default export from `main.wasp.ts`.
- Put declarations in `decls`: routes, pages, queries, actions, APIs, API namespaces, jobs, and CRUD.
- When `main.wasp.ts` references app components/functions, import them from `src/` with `with { type: "ref" }`.
- `@wasp.sh/spec` is generated by Wasp, don't install it from npm.
- Run `wasp compile` to check if app is valid.
- Do not run `tsc` directly for validation.
