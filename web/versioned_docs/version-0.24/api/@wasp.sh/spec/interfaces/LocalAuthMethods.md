# Interface: LocalAuthMethods

Built-in local auth methods.

Pick at most one of these.

## Properties

### email

> **email**: [`EmailAuthConfig`](EmailAuthConfig.md)

Enables email and password auth with email verification and password
reset flows.

***

### usernameAndPassword

> **usernameAndPassword**: [`UsernameAndPasswordConfig`](UsernameAndPasswordConfig.md)

Enables username and password auth.
